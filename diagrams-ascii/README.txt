SIMBOL YANG DIGUNAKAN:

BOXES (Kotak):
┌─────────┐
│         │  = Entitas, Proses, Aktivitas
└─────────┘

╔═════════╗
║         ║  = Header, Judul
╚═════════╝

ARROWS (Panah):
│  = Vertikal
─  = Horizontal
▼  = Ke bawah
▲  = Ke atas
►  = Ke kanan
◄  = Ke kiri

CONNECTIONS (Koneksi):
├─  = T-junction kiri
┤   = T-junction kanan
┬   = T-junction atas
┴   = T-junction bawah
┼   = Cross junction

DECISION (Keputusan):
    ╱────╲
   ╱      ╲
  ╱        ╲
 ◇          ◇  = Diamond shape
  ╲        ╱
   ╲      ╱
    ╲────╱

RELASI DATABASE:
│  = One
├─ = One (dengan cabang)
N  = Many
1  = One
FK = Foreign Key
PK = Primary Key
UK = Unique Key
