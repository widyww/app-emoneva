PENJELASAN ALUR:

1. LOGIN
   └─ Operator login menggunakan NPSN sebagai email dan password

2. DASHBOARD
   └─ Setelah login berhasil, masuk ke dashboard operator

3. INPUT IDENTITAS SEKOLAH
   ├─ NPSN (Nomor Pokok Sekolah Nasional)
   ├─ Nama sekolah
   ├─ Alamat lengkap
   ├─ Data kepala sekolah
   └─ Status akreditasi

4. INPUT SOSIAL EKONOMI BUDAYA
   ├─ Kondisi geografis sekolah
   ├─ Kondisi sosial ekonomi budaya
   └─ Akses transportasi

5. INPUT BANTUAN SEKOLAH
   ├─ Status penerimaan bantuan
   ├─ Nama lembaga pemberi bantuan
   └─ Keterangan detail bantuan

6. INPUT FASILITAS TIK
   ├─ Status dan sumber listrik
   ├─ Jumlah komputer
   ├─ Status dan sumber internet
   └─ Data laboratorium komputer

7. INPUT DATA GURU
   ├─ Identitas guru (NIP, NUPTK, dll)
   ├─ Kompetensi TIK
   ├─ Riwayat pelatihan
   └─ Kebutuhan pelatihan

8. VALIDASI KELENGKAPAN
   └─ Sistem cek apakah semua data sudah lengkap

9. AJUKAN VERIFIKASI
   ├─ Status berubah menjadi 'pending'
   └─ Notifikasi dikirim ke verifikator
